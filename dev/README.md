# Development

`index.html` at the repo root is generated by concatenating the files in `parts/`:

```sh
cat parts/01_head.html parts/02_core.js parts/03_ui.js > ../index.html
printf '</script>\n</body>\n</html>\n' >> ../index.html
```

`test.mjs` is a headless-browser smoke test (checks console errors, the eigen/determinant/rank
readouts, canvas dragging, presets, and the concept tours). To run it:

```sh
npm install playwright
node dev/test.mjs
```
